# StudyFlow 📚

StudyFlow is a simple study scheduler built in Python that helps students organize their subjects based on difficulty.

## 🚀 Features

* Add subjects with difficulty levels
* Calculate total study difficulty
* Organize study priorities
* Store study data using JSON

## 🛠️ Technologies

* Python
* JSON

## 📂 Project Structure

```
Studyflow/
│
├── main.py        # Entry point of the program
├── scheduler.py   # Study scheduling logic
├── storage.py     # Data storage and loading
├── data.json      # Stored study data
```

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/Pxfb/Studyflow.git
```

2. Run the project

```
python main.py
```
